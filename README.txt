У папці "Build" - знаходиться .apk файл для додатку на андроїд пристрої, створений на двигуні Unity(2021.3.12f1). 
У додатку реалізовано створення, видалення, реадгування, сортування за датою та типом (прибуток/витрати) транзакцій та кредитів. 
Також реалізовано перегляд курса валют(долар, євро, злотий), з використанням API: https://apilayer.com/marketplace/fixer-api#pricing

У папці "Hakaton_Economics" - знаходиться файли проєкту до цього додатка.
